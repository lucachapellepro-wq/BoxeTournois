# Guide d'utilisation — SAVATE BF

Application de gestion de tournoi de savate boxe française.

---

## Comment accéder à l'application

Ouvrez votre navigateur internet (Chrome, Firefox, Edge...) et tapez l'adresse qui vous a été communiquée.

L'application fonctionne aussi sur **téléphone et tablette**.

---

## Les 3 onglets principaux

En haut de l'écran, vous avez 3 boutons de navigation :

| Onglet | A quoi ça sert |
|--------|----------------|
| **Tournoi** | Créer et gérer vos tournois |
| **Tireurs** | Voir tous les tireurs inscrits, les modifier |
| **Clubs** | Gérer les clubs et leurs tireurs |

> Sur téléphone, appuyez sur le bouton **☰** en haut à droite pour voir les onglets.

---

## Etape 1 — Créer un club

Avant d'inscrire des tireurs, il faut au moins un club.

1. Allez dans l'onglet **Clubs**
2. Cliquez sur **+ Nouveau club**
3. Remplissez :
   - **Nom du club** (obligatoire)
   - **Ville** (obligatoire)
   - **Coach** (facultatif)
   - **Couleur** : cliquez sur un rond de couleur, ou utilisez le sélecteur de couleur pour choisir une couleur personnalisée. La couleur apparaîtra partout à côté du nom du club.
4. Cliquez sur **Créer le club**

### Modifier un club

1. Dans l'onglet **Clubs**, sélectionnez le club dans la liste déroulante
2. Cliquez sur **Modifier**
3. Changez les informations souhaitées
4. Cliquez sur **Sauvegarder**

---

## Etape 2 — Inscrire des tireurs

Vous pouvez inscrire des tireurs depuis l'onglet **Tireurs** ou depuis l'onglet **Clubs**.

1. Cliquez sur **+ Nouveau tireur**
2. Remplissez le formulaire :
   - **Nom** et **Prénom** (obligatoires)
   - **Sexe** : Homme ou Femme
   - **Type** : Tournoi ou Interclub
   - **Année de naissance** (facultatif mais recommandé)
   - **Poids** en kg (obligatoire)
   - **Gant** (grade) : Bleu, Vert, Rouge, Blanc, Jaune, Bronze, Argent ou Or
   - **Club** : choisissez dans la liste
3. Un **aperçu** apparaît en bas du formulaire avec la catégorie d'âge et de poids calculées automatiquement. Vérifiez que c'est correct !
4. Cliquez sur **Inscrire**

> Si un tireur avec le même nom et prénom existe déjà, un message d'erreur apparaît.

### Modifier un tireur dans le tableau

Dans l'onglet **Tireurs**, vous pouvez modifier directement dans le tableau :

- **Cliquez sur le nom, l'année, le poids ou le gant** d'un tireur pour le modifier
- Appuyez sur **Entrée** pour valider ou **Echap** pour annuler
- **Cliquez sur le badge Tournoi/Interclub** pour basculer entre les deux types

### Supprimer un tireur

- Cliquez sur l'icône **poubelle** à droite de la ligne
- Un message vert apparaît avec un bouton **Annuler** : si vous avez fait une erreur, cliquez dessus dans les 3 secondes pour restaurer le tireur !

---

## Etape 3 — Créer un tournoi

1. Allez dans l'onglet **Tournoi** (page d'accueil)
2. Cliquez sur **+ Nouveau tournoi**
3. Remplissez le **nom** et la **date**
4. Cliquez sur **Créer**

Votre tournoi apparaît sous forme de carte. Cliquez dessus pour l'ouvrir.

---

## Etape 4 — Ajouter des tireurs au tournoi

1. Ouvrez votre tournoi en cliquant sur sa carte
2. Cliquez sur **+ Ajouter des tireurs**
3. Une fenêtre s'ouvre avec la liste des tireurs disponibles
4. **Cherchez** un tireur par son nom ou son club dans la barre de recherche
5. Cliquez sur **+ Ajouter** à côté du tireur souhaité
6. Ou cliquez sur **Tout ajouter** pour inscrire tous les tireurs affichés d'un coup
7. Fermez la fenêtre quand vous avez terminé

### Changer le type d'un tireur (Tournoi / Interclub)

Dans la page du tournoi, cliquez directement sur le petit badge **T** ou **IC** à côté du nom d'un tireur. Le type change immédiatement.

### Retirer un tireur du tournoi

Cliquez sur le **✕** à côté du nom du tireur, puis confirmez.

---

## Etape 5 — Générer les combats

1. Dans la page du tournoi, cliquez sur **Affrontements**
2. Cliquez sur **Générer le tirage**
3. L'application crée automatiquement les combats :
   - **Elimination directe** (bracket) quand il y a assez de tireurs
   - **Poules** quand il y a 3 à 5 tireurs dans une catégorie
   - Les tireurs seuls dans leur catégorie deviennent des **vainqueurs directs**

### Les 4 onglets des affrontements

| Onglet | Contenu |
|--------|---------|
| **Elim. directe** | Les combats en arbre (huitièmes, quarts, demis, finale) |
| **Poules** | Les combats par poule (A, B, C...) avec demis et finale |
| **Interclub** | Les combats interclub, mixtes et manuels |
| **Vainqueurs** | Les tireurs qualifiés automatiquement (seuls en catégorie) |

### Ajouter un combat manuellement

1. Cliquez sur **+ Combat manuel**
2. Choisissez le premier tireur dans la liste
3. Choisissez le deuxième tireur
4. Le combat est créé immédiatement

> Vous pouvez aussi utiliser les filtres (recherche, catégorie) pour trouver plus facilement un tireur.

### Supprimer un combat manuel

Sur un combat ajouté manuellement, cliquez sur le **✕** en haut à droite de la carte. Un bouton **Annuler** apparaît dans la notification si vous changez d'avis.

### Régénérer les combats

Si vous avez ajouté ou retiré des tireurs et que vous voulez refaire le tirage :

1. Cliquez sur **Régénérer**
2. Confirmez (attention : tous les combats et résultats existants seront supprimés)

---

## Etape 6 — La feuille de tournoi (ordre des combats)

C'est la page à **imprimer le jour du tournoi**. Elle contient l'ordre des combats.

1. Dans les Affrontements, cliquez sur **Feuille**
2. Vous voyez le tableau des combats numérotés avec :
   - Le **type** de combat (couleur)
   - La **catégorie**
   - Le tireur **coin bleu** (à gauche)
   - Le tireur **coin rouge** (à droite)
   - Le club de chaque tireur (en couleur)

### Réorganiser l'ordre des combats

Vous pouvez **glisser-déposer** les lignes pour changer l'ordre :

1. Cliquez et maintenez sur une ligne de combat
2. Faites-la glisser vers la position souhaitée
3. Relâchez

### Espacement des combats

Le réglage **Espacement min** permet de s'assurer qu'un tireur ne combat pas deux fois de suite. Par exemple, avec un espacement de 2, il y aura au moins 2 combats entre deux passages du même tireur.

- Augmentez le chiffre pour plus d'espacement
- Cliquez sur **Randomiser** pour mélanger l'ordre en respectant l'espacement

### Légende des couleurs

En haut de la page, une légende indique la signification de chaque couleur :
- **Vert** = Poule
- **Violet** = Demi-finale
- **Rouge** = Finale
- **Bleu** = Elimination directe
- **Orange** = Interclub
- **Turquoise** = Interclub mixte
- **Orange foncé** = Combat ajouté

---

## Imprimer

Deux pages sont imprimables :

### Imprimer la feuille de tournoi
1. Allez dans **Feuille**
2. Cliquez sur **Imprimer**
3. Votre navigateur ouvre la fenêtre d'impression
4. Vérifiez que les couleurs sont activées (option "Graphiques d'arrière-plan" ou "Background graphics")

### Imprimer les affrontements
1. Allez dans **Affrontements**
2. Cliquez sur **Imprimer**
3. Tous les onglets (élimination, poules, interclub, vainqueurs) sont imprimés sur la même feuille

> **Astuce impression** : dans les paramètres d'impression, cochez **"Graphiques d'arrière-plan"** pour que les couleurs s'impriment correctement.

---

## Filtrer et rechercher (page Tireurs)

La page Tireurs dispose de filtres pour trouver rapidement un tireur :

- **Barre de recherche** : tapez un nom, prénom ou nom de club
- **Filtre par club** : sélectionnez un club dans la liste
- **Filtre H/F** : affichez uniquement les hommes ou les femmes
- **Filtre type** : Tournoi ou Interclub uniquement
- **Réinitialiser** : efface tous les filtres d'un coup

Les filtres se combinent entre eux. Le compteur en bas indique combien de tireurs correspondent.

---

## Trier le tableau des tireurs

Cliquez sur le **nom d'une colonne** pour trier :
- 1er clic : tri croissant (↑)
- 2ème clic : tri décroissant (↓)

Colonnes triables : Nom, Sexe, Type, Année, Poids, Gant, Cat. Poids, Cat. Âge, Club.

---

## Les catégories de poids et d'âge

Les catégories sont calculées **automatiquement** à partir du poids, du sexe et de l'année de naissance :

### Catégories d'âge
| Age | Catégorie |
|-----|-----------|
| 9 ans et moins | Pré-poussins |
| 10-11 ans | Poussins |
| 12-13 ans | Benjamins |
| 14-15 ans | Minimes |
| 16-17 ans | Cadets |
| 18-20 ans | Juniors |
| 21-34 ans | Seniors |
| 35-39 ans | Vétérans Combat |
| 40 ans et plus | Vétérans |

### Grades de gant
| Gant | Degré |
|------|-------|
| Bleu | 1er degré |
| Vert | 2ème degré |
| Rouge | 3ème degré |
| Blanc | 4ème degré |
| Jaune | 5ème degré |
| Bronze | — |
| Argent | — |
| Or | — |

## En cas de problème

- **J'ai supprimé un tireur par erreur** : cliquez immédiatement sur **Annuler** dans la notification verte qui apparaît en bas.
- **Les combats ne sont pas bons** : cliquez sur **Régénérer** dans la page Affrontements pour refaire le tirage.
- **L'impression n'a pas de couleurs** : activez l'option "Graphiques d'arrière-plan" dans les paramètres d'impression de votre navigateur.
- **La page ne se charge pas** : rafraîchissez la page (touche F5 ou bouton actualiser du navigateur).
- **Je ne trouve pas un tireur** : vérifiez vos filtres dans la page Tireurs et cliquez sur **Réinitialiser** si besoin.
